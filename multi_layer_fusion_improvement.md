# 多层特征融合改进方案：Z-score 归一化 + Top-k/Max 聚合

## 问题诊断

当前做法：将 DINOv3 第 5–11 层的逐层异常分数图 $s_\ell(p)$ 直接相加得到最终异常分数。

直接相加的三个隐含错误假设：

1. **尺度不可比**：深层 cosine 距离方差远大于浅层，求和后浅层信号被吞没。
2. **噪声分布不同**：不同层对光照残差、配准误差的敏感度不同，相加等于把噪声也混进来。
3. **异常类型不同质**：missing-part 型异常在 layer 5 响应强，scratch 型在 layer 7–9 强。统一权重相加意味着"某层强烈响应"和"所有层弱响应"被错误等价。

## 改进方案

### Step 1：跨层 Z-score 归一化

**目的**：将所有层的异常分数映射到同一尺度。

**做法**：

- 对每个物体，用训练集中的正常 ref-query 对（同物体、同视角、不同光照、无异常），计算每层每个 patch 位置的差异分数。
- 统计每层 $\ell$ 的全局均值 $\mu_\ell$ 和标准差 $\sigma_\ell$（可以所有 patch 位置一起统计，也可以按空间位置分别统计）。
- 推断时做归一化：

$$\tilde{s}_\ell(p) = \frac{s_\ell(p) - \mu_\ell}{\sigma_\ell}$$

**注意事项**：

- $\mu_\ell, \sigma_\ell$ 的估计不需要异常样本，纯用正常对即可。
- 这一步是离线统计，只需跑一次，可以直接写在硬盘中（实验所在文件夹下）。
- 配置文档中输入--Z-score，启用


### Step 2：Top-k 聚合替换 Sum


$$S(p) = \frac{1}{k} \sum_{\ell \in \text{top-}k(\tilde{s}(p))} \tilde{s}_\ell(p)$$
